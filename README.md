Проект демонстрирует работу с библиотеками Pandas, NumPy и Scikit-learn для анализа данных о автомобилях и предсказания их стоимости. Основная задача — построить модель линейной регрессии для предсказания цены автомобиля (MSRP) и сравнить её точность с «глупой моделью», которая всегда предсказывает среднюю цену.
